Eine auf Inkscape basierende, quelloffene Design-Plattform für Stickereien.

Ink/Stitch hat sich zum Ziel gesetzt, eine vollwertige Digitalisierungssoftware zu sein, mit komplett freier Software (Open Source). Wir wollen eine einfach zugängliche Software bereitstellen, die sowohl für Hobbyisten geeignet ist, als auch professionellen Designern alle Funktionen bietet, die sie benötigen.
